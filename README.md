# 🎬 Squid Game Landing Page

A modern and cinematic landing page inspired by the *Squid Game* series. Built using HTML and CSS with a focus on visual effects, layout design, and UI styling.

---

## 📸 Preview

This project includes:

* Hero section with character background
* Animated UI-style layout
* Episode cards section
* Glassmorphism effects
* Social media icons
* Scroll indicator design
* Fully responsive layout

---

## 🚀 Features

* Modern dark UI design
* Gradient background effects
* Hover animations on cards
* Glassmorphism episode section
* Responsive layout for mobile devices
* Stylish typography and spacing
* Smooth UI interactions

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* Flexbox
* Grid Layout
* Google Fonts (Inter)
* Remix Icons
